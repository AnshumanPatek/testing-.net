# Gainateliê CMS

Custom CMS built with .NET backend and Next.js frontend for managing the Gainateliê website.

## Tech Stack

**Backend:**
- .NET 8.0
- SQL Server
- Entity Framework Core
- ASP.NET Core Web API
- AWS S3 for media storage

**Frontend:**
- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS

## Project Structure

```
/backend          - .NET Web API
/frontend         - Next.js public website
/admin            - Next.js admin panel
/database         - SQL Server schema
```

## Features

### Public Website
- Responsive landing page with all sections
- Project case studies with dynamic pages
- 404 and Privacy Policy pages
- SEO optimized

### Admin Panel
- Role-based access (Admin, CRM, Editor)
- Google Analytics dashboard
- Real-time page editor with preview
- Block-based project page constructor
- Media library with S3 integration
- Draft/publish workflow
